Welcome

This is a proof-of-concept app for issuing diplomas for students as NFTs.

This project assumes that you connect with the college's crypto wallet and issue the diplomas to students' wallets which you have collected from them beforehand.

All of the diplomas and money used here are not real and are not ment to be real. Please don't sue me for forgering valuable documents, thank you.
